Sistema de Gestão Escolar (SGE) - Instituto Recriar

Este projeto representa a interface de usuário (Front-end) de um Sistema de Gestão Escolar, desenvolvido para facilitar a administração de alunos, professores, turmas e aulas em uma instituição de ensino.

Ideia Central do Projeto

A ideia principal é fornecer uma plataforma centralizada e intuitiva para que administradores e professores possam gerenciar as atividades acadêmicas e administrativas. O sistema é projetado para ser modular, permitindo a fácil expansão de funcionalidades como relatórios, controle de frequência e gestão de usuários.

Arquitetura Técnica

O Front-end é construído utilizando **HTML5, CSS3 e JavaScript puro**, seguindo o conceito de **Single Page Application (SPA)**, onde a navegação entre as páginas é simulada pelo carregamento de conteúdo dinâmico (embora neste caso, a navegação seja feita entre arquivos HTML separados, a lógica de autenticação e comunicação com a API é centralizada).

A comunicação com o Back-end é realizada através de requisições **RESTful** (GET, POST, PUT, DELETE) para a URL base: http://137.131.197.181:8080/api. A autenticação é baseada em **Token Bearer** armazenado no sessionStorage.

Módulos Principais

  • **Gestão de Usuários:** Controle de acesso e permissões (Administradores e Professores).
  • **Gestão de Alunos:** Cadastro, busca, edição e exclusão de informações dos estudantes.
  • **Gestão de Professores:** Cadastro e administração do corpo docente.
  • **Gestão de Turmas:** Criação, alteração e associação de alunos e professores às turmas.
  • **Gestão de Aulas:** Registro e acompanhamento das aulas.

O sistema visa a eficiência e a organização dos dados escolares, proporcionando uma experiência de usuário limpa e responsiva.
