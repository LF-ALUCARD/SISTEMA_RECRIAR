Informações do Sistema

Visão Geral do Projeto

O **Sistema de Gestão Escolar (SGE) - Instituto Recriar** é a interface de usuário (Front-end) desenvolvida para facilitar a administração de alunos, professores, turmas e responsáveis na instituição.

O objetivo é fornecer uma plataforma centralizada e intuitiva para a gestão eficiente das atividades acadêmicas e administrativas.

Arquitetura Técnica

O Front-end é construído com **HTML5, CSS3 e JavaScript puro**, seguindo um design modular e responsivo.

Comunicação com o Back-end

A comunicação é feita via requisições **RESTful** (GET, POST, PUT, DELETE) para a URL base:

${CONFIG.API_BASE_URL}

A autenticação é baseada em **Token Bearer** armazenado no sessionStorage.

Módulos Principais

  • **Gestão de Usuários:** Controle de acesso e permissões (Administradores).
  • **Gestão de Alunos:** Cadastro, busca, edição e exclusão de estudantes.
  • **Gestão de Professores:** Cadastro e administração do corpo docente.
  • **Gestão de Turmas:** Criação, alteração e associação de alunos e professores.
  • **Gestão de Responsáveis:** Listagem e exclusão de responsáveis pelos alunos.

Próximos Passos Sugeridos

Para expandir o sistema, sugerimos:

  • Implementar páginas de cadastro/edição para Responsáveis.
  • Adicionar funcionalidades de Relatórios e controle de Frequência.
  • Melhorar a gestão de usuários com mais níveis de acesso.
